# ⏳ Python Countdown Timer

A Python program that counts down from a user-defined time and displays the remaining time in a formatted way with colored terminal output.

## 🚀 Features
- Real-time countdown timer
- Displays time in minutes and seconds (MM:SS)
- Colored terminal output 🎨
- Simple and user-friendly interface
- Beginner-friendly code

## 🛠 Built With
- Python
- time module
- Colorama

## 📦 Installation

Install required library:

pip install colorama

## ▶️ How to Run

1. Download or clone the repository  
2. Open the project folder  
3. Run the program  

python countdown_timer.py
